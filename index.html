stochseq

Statistical Inference for Nanopore Sequencing with a Biased Random Walk Model

This repository includes files and scripts used in the following paper:

K. Emmett, J. Rosenstein, J.W. van de Meent, K. Shepard, and C. Wiggins. Stochastic Inference For Nanopore Sequencings with a Biased Random Walk Model. Biophysical Journal. 2015.

At present, the model is implemented in matlab. To run:

  • stochseq_build(seqlength, bias, err, nreads) : builds synthetic data
  • stochseq_infer(model) : performs inference on model object

Functions:

  • gen_dna(L) : generates a random dna sequence where each base is represented by an integer between 1 and 4
  • gen_read(dna, bias, err) : generates a random walk along 'dna'

Parameters:

  • seqlength : sequence length
  • bias : forward bias
  • err : per base call error rate
  • nreads : number of reads
